# cloud-learning-ask-service 问答服务

## 概述

问答服务主要负责处理问答相关的业务功能，包括问题管理、回答管理和分类管理等。

## 问题管理接口

### 1. 提出问题

- **接口路径**: `POST /auth-api/question`
- **接口描述**: 提出一个新问题
- **权限要求**: 需要登录认证
- **请求参数**:
```json
{
  "title": "问题标题",           // 必填，问题标题
  "content": "问题内容",         // 必填，问题详细内容
  "image": "题图URL",          // 可选，问题配图URL
  "cidList": [1, 2, 3]        // 可选，问题所属分类ID列表
}
```
- **响应参数**:
```json
{
  "id": 1,                    // 问题ID
  "title": "问题标题",         // 问题标题
  "content": "问题内容",       // 问题内容
  "image": "题图URL",         // 问题配图URL
  "memberId": 1001,           // 问题发布者ID
  "status": "published",      // 问题状态
  "favoriteNum": 10,          // 收藏数量
  "likeNum": 20,              // 点赞数量
  "commentNum": 5,            // 评论数量
  "member": {                 // 问题发布者信息
    "id": 1001,
    "nickname": "用户昵称",
    "avatar": "头像URL"
  },
  "like": {                   // 当前用户点赞状态
    "isLike": true
  },
  "watchNum": 100,            // 浏览数量
  "cidList": [1, 2, 3],      // 问题所属分类ID列表
  "categoryList": [           // 问题所属分类信息
    {
      "id": 1,
      "name": "分类名称"
    }
  ],
  "answerNum": 5,             // 回答数量
  "createTime": "2023-05-01 12:00:00",  // 创建时间
  "updateTime": "2023-05-02 13:00:00"   // 更新时间
}
```

### 2. 修改问题

- **接口路径**: `PUT /auth-api/question`
- **接口描述**: 修改已有问题
- **权限要求**: 需要登录认证
- **请求参数**:
```json
{
  "id": 1,                    // 必填，问题ID
  "title": "修改后的问题标题", // 可选，修改后的问题标题
  "content": "修改后的问题内容", // 可选，修改后的问题内容
  "image": "修改后的题图URL", // 可选，修改后的问题配图URL
  "status": "published",     // 可选，问题状态
  "cidList": [1, 2, 3]       // 可选，修改后的问题所属分类ID列表
}
```
- **响应参数**: 同提出问题的响应参数

### 3. 删除问题

- **接口路径**: `DELETE /auth-api/question`
- **接口描述**: 删除一个问题
- **权限要求**: 需要登录认证
- **请求参数**:
```json
{
  "id": 1                     // 必填，要删除的问题ID
}
```
- **响应参数**: 无返回数据

### 4. 获取问题列表

- **接口路径**: `GET /question/list`
- **接口描述**: 获取问题列表（需要权限）
- **权限要求**: 需要权限验证
- **请求参数**:
```json
{
  "pageNum": 1,               // 可选，页码，默认为1
  "pageSize": 10,             // 可选，每页大小，默认为10
  "keyword": "搜索关键字",      // 可选，搜索关键字
  "status": "published",      // 可选，问题状态
  "cid": 1,                   // 可选，分类ID
  "memberId": 1001,           // 可选，会员ID
  "idList": [1, 2, 3],        // 可选，问题ID列表
  "orderColumn": "createTime", // 可选，排序字段，可选值：answerNum(回答数), createTime(创建时间), watchNum(浏览数)
  "orderDirection": "desc"    // 可选，排序方向，asc或desc
}
```
- **响应参数**:
```json
{
  "pageNum": 1,               // 当前页码
  "pageSize": 10,             // 每页大小
  "total": 100,               // 总记录数
  "list": [                   // 问题列表
    {
      // 同问题详情响应参数
    }
  ]
}
```

### 5. 获取问题列表（无权限）

- **接口路径**: `GET /public-api/question/list`
- **接口描述**: 获取问题列表（无需权限）
- **权限要求**: 无需权限
- **请求参数**: 同获取问题列表
- **响应参数**: 同获取问题列表

### 6. 获取问题详情

- **接口路径**: `GET /public-api/question`
- **接口描述**: 获取问题详情
- **权限要求**: 无需权限
- **请求参数**:
```json
{
  "id": 1                     // 必填，问题ID
}
```
- **响应参数**: 同提出问题的响应参数

### 7. 获取会员问题数量

- **接口路径**: `GET /public-api/question/member/count`
- **接口描述**: 获取指定会员的问题数量
- **权限要求**: 无需权限
- **请求参数**:
```json
{
  "memberId": 1001            // 可选，会员ID，不填则获取当前登录用户的问题数量
}
```
- **响应参数**:
```json
{
  "count": 10                 // 问题数量
}
```

### 8. 获取会员问题列表

- **接口路径**: `GET /auth-api/member/question/list`
- **接口描述**: 获取指定会员的问题列表
- **权限要求**: 需要登录认证
- **请求参数**: 同获取问题列表，memberId为可选参数，不填则获取当前登录用户的问题列表
- **响应参数**: 同获取问题列表

```json
}
```
- **响应参数**:
```json
{

  },
```

## 分类管理接口

### 1. 获取分类列表（管理员）

- **接口路径**: `GET /category/admin/list`
- **接口描述**: 获取分类列表（管理员视角）
- **权限要求**: 需要管理员权限
- **请求参数**:
```json
{
  "isShow": true,             // 可选，是否显示
  "isShowIndex": true         // 可选，是否在首页显示
}
```
- **响应参数**:
```json
[
  {
    "id": 1,                  // 分类ID
    "name": "分类名称",        // 分类名称
    "sortOrder": 1,           // 排序序号
    "isShow": true,           // 是否显示
    "isShowIndex": true,      // 是否在首页显示
    "image": "分类图片URL",   // 分类图片
    "level": 1,               // 分类级别
    "pid": 0,                 // 父级ID
    "createTime": "2023-05-01 12:00:00",  // 创建时间
    "updateTime": "2023-05-02 13:00:00"   // 更新时间
  }
]
```

### 2. 获取分类详情

- **接口路径**: `GET /category/{id}`
- **接口描述**: 获取指定分类的详情
- **权限要求**: 无需权限
- **请求参数**:
```json
{
  "id": 1                     // 必填，分类ID，通过路径参数传递
}
```
- **响应参数**: 同获取分类列表中的单个分类对象

### 3. 添加分类

- **接口路径**: `POST /category`
- **接口描述**: 添加一个新的分类
- **权限要求**: 需要管理员权限

```json
{
```

### 4. 修改分类

## 数据模型

### 问题状态枚举 (QuestionStatus)

- `draft`: 草稿
- `published`: 已发布
- `closed`: 已关闭

### 排序字段枚举 (QuestionPageRequest.OrderColumn)

- `answerNum`: 回答数量
- `createTime`: 创建时间
- `watchNum`: 浏览数量

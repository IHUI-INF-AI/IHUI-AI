# 视频生成

## 功能介绍

视频生成功能支持使用AI生成视频，包括Kling AI和One-Click视频生成。

## 使用方式

### Kling AI视频生成

1. 进入视频生成页面
2. 选择"Kling AI"标签
3. 上传视频或图片
4. 识别视频中的人脸（可选）
5. 输入视频生成提示词
6. 选择人脸ID（如果识别了人脸）
7. 点击"生成视频"
8. 等待生成完成

### One-Click视频生成

1. 选择"One-Click"标签
2. 输入视频主题
3. 选择视频时长
4. 选择视频比例
5. 选择音频风格
6. 点击"生成视频"
7. 等待生成完成

## 功能特点

- **人脸识别** - 自动识别视频中的人脸
- **人脸替换** - 使用指定人脸生成视频
- **首帧图片** - 从提示词或上一视频的最后一帧生成首帧图片
- **状态查询** - 实时查询视频生成状态
- **自动轮询** - 自动检查生成状态
- **视频预览** - 预览生成的视频
- **视频下载** - 下载生成的视频

## 使用技巧

1. **清晰提示词** - 使用清晰、具体的提示词描述视频内容
2. **选择合适时长** - 根据需求选择合适的视频时长
3. **等待生成** - 视频生成需要时间，请耐心等待
4. **保存作品** - 生成完成后及时保存到作品画廊

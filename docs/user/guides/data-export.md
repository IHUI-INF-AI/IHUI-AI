# 数据导出

## 功能介绍

数据导出功能支持将数据导出为多种格式，包括CSV、Excel、PDF和批量ZIP。

## 使用方式

### CSV导出

1. 在数据列表页面
2. 点击"导出"按钮
3. 选择"CSV格式"
4. 确认导出
5. 下载文件

### Excel导出

1. 点击"导出"按钮
2. 选择"Excel格式"
3. 确认导出
4. 下载Excel文件

### PDF导出

1. 点击"导出"按钮
2. 选择"PDF格式"
3. 确认导出
4. 下载PDF文件

### 批量导出

1. 选择多个数据表
2. 点击"批量导出"
3. 选择导出格式
4. 确认导出
5. 下载ZIP压缩包

## 支持的数据

- **交易记录** - 钱包交易记录
- **订单列表** - 订单数据
- **预约记录** - 服务预约记录
- **其他数据** - 支持导出的其他数据

## 使用技巧

1. **筛选数据** - 导出前先筛选需要的数据
2. **选择格式** - 根据用途选择合适的格式
3. **批量导出** - 需要导出多个表时使用批量导出
4. **定期备份** - 定期导出重要数据作为备份

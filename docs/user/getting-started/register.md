# 注册与登录

## 注册账号

### 方式一：手机号注册

1. 访问登录页面
2. 选择"手机号登录"
3. 输入手机号
4. 获取并输入验证码
5. 设置密码（可选）
6. 完成注册

### 方式二：邮箱注册

1. 访问登录页面
2. 选择"邮箱注册"
3. 输入邮箱地址
4. 获取并输入验证码
5. 设置密码
6. 完成注册

### 方式三：第三方登录

支持以下第三方登录方式：

- **微信登录** - 使用微信扫码登录
- **支付宝登录** - 使用支付宝扫码登录
- **飞书登录** - 使用飞书扫码登录

## 登录账号

### 手机号登录

1. 输入手机号
2. 获取验证码
3. 输入验证码登录

或使用密码登录：

1. 输入手机号
2. 输入密码
3. 点击登录

### 邮箱登录

1. 输入邮箱地址
2. 输入密码
3. 点击登录

### 第三方登录

1. 选择第三方登录方式
2. 扫码或授权
3. 自动登录

## 忘记密码

1. 在登录页面点击"忘记密码"
2. 输入手机号或邮箱
3. 获取验证码
4. 设置新密码
5. 完成重置

## 安全建议

- 使用强密码（包含大小写字母、数字、特殊字符）
- 定期修改密码
- 不要分享账号信息
- 启用双因素认证（如果可用）

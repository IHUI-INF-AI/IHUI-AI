# 个人中心

## 个人信息管理

### 修改头像

1. 进入个人中心
2. 点击头像区域
3. 选择或上传新头像
4. 确认保存

### 修改昵称

1. 进入个人中心
2. 点击昵称编辑按钮
3. 输入新昵称
4. 保存更改

### 修改邮箱

1. 进入个人中心
2. 找到邮箱设置
3. 输入新邮箱
4. 验证邮箱
5. 完成修改

### 修改手机号

1. 进入个人中心
2. 找到手机号设置
3. 输入新手机号
4. 验证手机号
5. 完成修改

## 账户安全

### 修改密码

1. 进入个人中心 > 账户安全
2. 点击"修改密码"
3. 输入当前密码
4. 输入新密码
5. 确认新密码
6. 完成修改

### 登录记录

- 查看最近登录记录
- 查看登录设备信息
- 异常登录提醒

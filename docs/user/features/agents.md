# 智能体使用

## 功能介绍

智能体是预配置的AI助手，可以执行特定任务或提供专业服务。

## 使用方式

### 查找智能体

1. 进入智能体市场
2. 浏览智能体列表
3. 使用筛选功能（分类、标签、价格等）
4. 搜索智能体

### 使用智能体

1. 点击智能体卡片查看详情
2. 点击"使用"或"立即使用"
3. 在对话界面与智能体交互
4. 按照智能体的提示操作

### 购买智能体

1. 查看智能体价格和功能
2. 确认购买
3. 选择支付方式
4. 完成支付后即可使用

## 智能体类型

### 免费智能体

- 基础功能免费使用
- 适合日常使用
- 功能相对简单

### 付费智能体

- 专业功能
- 高级能力
- 定制化服务

## 使用技巧

1. **阅读说明** - 使用前仔细阅读智能体的功能说明
2. **提供信息** - 按照智能体的要求提供必要信息
3. **反馈问题** - 遇到问题及时反馈
4. **收藏常用** - 收藏常用的智能体，方便快速访问

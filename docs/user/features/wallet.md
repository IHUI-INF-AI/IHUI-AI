# 钱包功能

## 账户余额

钱包显示您的账户余额，包括：

- **总余额** - 账户总金额
- **可用余额** - 可使用的金额
- **冻结余额** - 冻结中的金额

## 充值

### 充值步骤

1. 进入钱包页面
2. 点击"充值"按钮
3. 选择充值金额
4. 选择支付方式（微信、支付宝）
5. 完成支付
6. 等待到账（通常即时到账）

### 充值记录

- 查看所有充值记录
- 查看充值状态（成功、失败、处理中）
- 查看充值详情

## 提现

### 提现步骤

1. 进入钱包页面
2. 点击"提现"按钮
3. 输入提现金额
4. 选择提现方式
5. 绑定提现账户（首次需要）
6. 提交提现申请
7. 等待审核和处理

### 提现记录

- 查看所有提现记录
- 查看提现状态（审核中、处理中、成功、失败）
- 查看提现详情

## 交易记录

### 查看记录

1. 进入钱包页面
2. 点击"交易记录"
3. 查看所有交易

### 筛选功能

- 按类型筛选（收入、支出、全部）
- 按状态筛选
- 按时间范围筛选
- 关键词搜索

### 交易详情

- 交易类型
- 交易金额
- 交易时间
- 交易状态
- 交易备注

## 统计分析

### 统计图表

- 收入/支出趋势图
- 分类统计饼图
- 时间范围选择（今日、本周、本月、本年、全部）

### 统计指标

- 总收入
- 总支出
- 净收入
- 分类统计

## 导出功能

支持导出交易记录：

- **CSV格式** - 适合Excel打开
- **Excel格式** - 包含格式和样式
- **PDF格式** - 适合打印和分享

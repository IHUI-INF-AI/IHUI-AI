# 支付功能

## 支付方式

平台支持以下支付方式：

### 平台币支付

- 使用账户余额支付
- 即时到账
- 无需跳转

### 微信支付

- 扫码支付
- 安全可靠
- 实时到账

### 支付宝支付

- 扫码支付
- 安全可靠
- 实时到账

## 支付流程

### 创建订单

1. 选择商品或服务
2. 确认订单信息
3. 选择支付方式
4. 创建订单

### 完成支付

1. 根据支付方式操作：
   - **平台币** - 直接扣除余额
   - **微信/支付宝** - 扫码支付
2. 等待支付结果
3. 支付成功后自动跳转

### 支付状态

- **待支付** - 订单已创建，等待支付
- **支付中** - 正在处理支付
- **支付成功** - 支付完成
- **支付失败** - 支付失败，可重试
- **已取消** - 订单已取消

## 订单管理

### 查看订单

- 进入订单列表
- 查看订单详情
- 查看支付状态

### 订单操作

- **重新支付** - 支付失败的订单可以重新支付
- **取消订单** - 未支付的订单可以取消
- **申请退款** - 已支付的订单可以申请退款

## 支付安全

- 所有支付均通过安全通道
- 支持HTTPS加密
- 不存储支付密码
- 支付状态实时同步

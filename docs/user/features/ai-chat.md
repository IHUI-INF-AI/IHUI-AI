# AI对话

## 功能介绍

AI对话功能允许您与多种大语言模型进行智能对话，支持流式响应和上下文管理。

## 使用方式

### 开始对话

1. 进入AI对话页面
2. 选择AI模型（GPT-4、Claude、Gemini等）
3. 输入您的问题或需求
4. 点击发送或按Enter键

### 对话功能

- **多轮对话** - 支持上下文记忆，可以进行连续对话
- **流式响应** - 实时显示AI生成的内容
- **消息编辑** - 可以编辑已发送的消息
- **消息删除** - 可以删除不需要的消息
- **对话历史** - 自动保存对话记录

### 高级功能

- **引用回复** - 可以引用之前的消息进行回复
- **消息搜索** - 在对话历史中搜索消息
- **导出对话** - 导出对话记录为文本或Markdown格式
- **语音输入** - 支持语音转文字输入（如果浏览器支持）

## 模型选择

平台支持以下AI模型：

- **GPT-4** - OpenAI最新模型，适合复杂任务
- **GPT-3.5** - 快速响应，适合日常对话
- **Claude** - Anthropic模型，擅长分析和推理
- **Gemini** - Google模型，多模态能力
- **其他模型** - 根据需求选择

## 使用技巧

1. **清晰表达** - 尽量清晰地描述您的需求
2. **提供上下文** - 在对话中提供必要的背景信息
3. **迭代优化** - 根据AI的回答，逐步优化您的问题
4. **使用引用** - 引用之前的消息，帮助AI理解上下文

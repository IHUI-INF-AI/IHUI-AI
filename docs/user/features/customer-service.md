# 客服系统

## 在线客服

### 开始对话

1. 进入客服系统页面
2. 选择"在线客服"标签
3. 开始与客服对话

### 发送消息

- **文本消息** - 直接输入文字
- **图片消息** - 上传图片
- **文件消息** - 上传文件

### 快捷回复

- 查看常见问题
- 使用快捷回复按钮
- 快速解决问题

## 服务工单

### 创建工单

1. 进入客服系统页面
2. 选择"服务工单"标签
3. 点击"创建工单"
4. 填写工单信息：
   - 工单类型
   - 问题标题
   - 问题描述
   - 附件（可选）
5. 提交工单

### 查看工单

- 查看所有工单
- 查看工单详情
- 查看工单回复
- 查看工单状态

### 工单状态

- **待处理** - 工单已提交，等待处理
- **处理中** - 客服正在处理
- **已解决** - 问题已解决
- **已关闭** - 工单已关闭

### 工单操作

- **回复工单** - 对工单进行回复
- **关闭工单** - 关闭已解决的工单
- **重新打开** - 重新打开已关闭的工单
- **评价工单** - 对工单服务进行评价

## 常见问题

### 查看常见问题

1. 进入客服系统页面
2. 选择"常见问题"标签
3. 浏览问题分类
4. 查看问题答案

### 问题分类

- 账户相关
- 支付相关
- 功能使用
- 技术问题
- 其他问题

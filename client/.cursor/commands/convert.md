# 代码转换

在不同语言/框架之间转换代码。

## 指令

请将以下代码转换为目标格式：

{{selection}}

### 支持的转换

1. **JavaScript ↔ TypeScript**
   - 添加/移除类型注解
   - 处理类型推断

2. **Vue 2 → Vue 3**
   - Options API → Composition API
   - this.xxx → ref/reactive
   - 生命周期钩子更新

3. **Class → Function**
   - 类组件 → 函数组件
   - 类 → 工厂函数

4. **CSS ↔ Tailwind**
   - 传统 CSS → Tailwind 类
   - Tailwind → 传统 CSS

5. **其他转换**
   - JSON ↔ YAML
   - Promise → async/await
   - CommonJS → ES Modules

### 转换原则

- 保持功能完全一致
- 遵循目标格式的最佳实践
- 保留必要的注释
- 优化可以优化的部分

### 输出

1. 转换后的完整代码
2. 转换说明（重要变更）
3. 注意事项（如有）

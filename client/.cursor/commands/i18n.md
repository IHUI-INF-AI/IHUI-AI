# 国际化处理

提取文本并生成 i18n 配置。

## 指令

请处理以下代码的国际化：

{{selection}}

### 处理步骤

1. **提取硬编码文本**
   - 识别所有中文/英文硬编码
   - 生成语义化的 key

2. **替换为 i18n 调用**
```vue
<!-- 模板中 -->
<template>
  <!-- 之前 -->
  <span>用户名</span>
  
  <!-- 之后 -->
  <span>{{ $t('user.username') }}</span>
</template>

<!-- JS 中 -->
<script setup>
import { useI18n } from 'vue-i18n'
const { t } = useI18n()

// 之前
const message = '操作成功'

// 之后
const message = t('common.success')
</script>
```

3. **生成语言文件**
```typescript
// zh-CN.ts
export default {
  common: {
    success: '操作成功',
    error: '操作失败',
    confirm: '确认',
    cancel: '取消',
  },
  user: {
    username: '用户名',
    password: '密码',
  },
}

// en-US.ts
export default {
  common: {
    success: 'Operation successful',
    error: 'Operation failed',
    confirm: 'Confirm',
    cancel: 'Cancel',
  },
  user: {
    username: 'Username',
    password: 'Password',
  },
}
```

### 输出

1. 修改后的组件代码
2. 中文语言文件内容
3. 英文语言文件内容

# 翻译代码注释

将代码中的注释在中英文之间互相翻译。

## 指令

请翻译以下代码中的注释：

{{selection}}

### 翻译规则

1. **自动检测语言**
   - 中文注释 → 翻译为英文
   - 英文注释 → 翻译为中文

2. **保持格式**
   - 保持原有的注释格式（单行/多行）
   - 保持代码缩进不变
   - 不修改代码逻辑

3. **翻译质量**
   - 使用专业的技术术语
   - 保持简洁清晰
   - 符合目标语言的表达习惯

### 示例

输入：
```javascript
// 获取用户列表
const users = await fetchUsers()
```

输出：
```javascript
// Get user list
const users = await fetchUsers()
```

### 输出要求

- 直接输出翻译后的完整代码
- 只翻译注释，不修改代码
- 保持原有的代码格式

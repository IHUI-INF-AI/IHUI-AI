### 检查项目

1. **XSS 攻击**
   - innerHTML 使用
   - v-html 指令
   - 用户输入未转义

2. **SQL 注入**
   - 字符串拼接 SQL
   - 未使用参数化查询

3. **CSRF 攻击**
   - Token 验证
   - SameSite Cookie

4. **敏感信息泄露**
   - 硬编码密码/密钥
   - console.log 输出敏感数据
   - 错误信息暴露

5. **认证授权**
   - 权限检查缺失
   - JWT 处理不当
   - Session 管理

6. **前端安全**
   - localStorage 存储敏感数据
   - 第三方脚本风险
   - 点击劫持防护

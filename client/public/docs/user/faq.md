# 常见问题

## 账户相关

### 如何注册账号？

支持以下注册方式：
- 手机号注册
- 邮箱注册
- 第三方登录（微信、支付宝、飞书）

### 忘记密码怎么办？

1. 在登录页面点击"忘记密码"
2. 输入手机号或邮箱
3. 获取验证码
4. 设置新密码

### 如何修改个人信息？

进入个人中心，可以修改：
- 头像
- 昵称
- 邮箱
- 手机号
- 密码

## 支付相关

### 支持哪些支付方式？

- 平台币（账户余额）
- 微信支付
- 支付宝支付

### 充值多久到账？

通常即时到账，如有延迟请联系客服。

### 如何申请提现？

1. 进入钱包页面
2. 点击"提现"
3. 输入提现金额
4. 选择提现方式
5. 提交申请
6. 等待审核

## 功能使用

### 如何使用AI对话？

1. 进入AI对话页面
2. 选择AI模型
3. 输入问题
4. 等待回答

### 如何生成视频？

支持两种方式：
- Kling AI视频生成
- One-Click视频生成

详细说明请参考"视频生成"文档。

### 如何预约技术服务？

1. 进入技术服务专区
2. 选择服务类型
3. 填写预约信息
4. 提交预约

本文档说明智汇开放平台（iHui Open Platform）在 GitHub 上的半开源分发方式、定价、售卖规则与开源协议。适用于企业采购、二次开发与合规审计参考。

**与官网一致**：官网开放平台页（`/open`）售卖区定价、许可与退款等文案与本 README 保持一致，具体展示以 i18n（`openPlatform.sale.*`、`openPlatform.pricingCompare.*` 等）与页面为准。与 BACKLOG 九、9.2 同步：定价表、FAQ、联系/申请入口以官网与本文档为准。

## 一、产品说明

智汇开放平台以**半开源（Semi-Open Source）**形式提供：

- **开放部分**：核心前端、开放能力接口与文档可被购买者**查看、学习、修改**，并在约定范围内**内部使用与部署**。
- **保留部分**：部分后端服务、商业模块与专有算法保留闭源，不随源码包提供。

适用场景包括但不限于：

- 企业需要将 AI 能力集成到自有系统并做二次开发
- 需要源码级审计与合规审查
- 希望在内网或私有云环境部署
- 需要与现有工作流（n8n、Dify 等）深度集成

## 三、售卖规则

### 3.1 售卖渠道

- **GitHub**：在指定仓库通过 Release 发布可购买版本，购买后获得私有仓库访问权或源码包下载链接。
- **官网/商务**：通过官网留资或与销售签订合同，支持对公转账与合同流程。

### 3.2 支付与发票

- 支持对公转账、在线支付（以实际开通为准）。
- 可开具增值税普通发票或增值税专用发票，开票信息以签约时确认为准。

### 3.3 授权范围

- 购买后您获得**对应版本**的源码访问权与**许可使用权**。
- **允许**：在自有业务中查看、学习、修改、内部部署及在约定范围内使用。
- **禁止**：将源码或衍生作品向第三方再分发、转售，或用于与智汇 AI 直接竞争的产品。

### 3.4 退款政策

- 数字商品（源码、文档、访问权）一经交付，原则上**不退款**。
- 如有特殊情形（如未获得约定交付物、重大瑕疵），可与客服协商，最终以合同或平台规则为准。

### 3.5 续期与升级

- **订阅制**：按年续费，到期前会提醒续费；逾期未续费将失去访问与更新权限。
- **买断制**（若提供）：在约定周期内可获得 Bug 修复与安全更新；大版本升级可能需单独议价。

## 四、开源协议（许可证）

### 4.1 许可证名称

**智汇开放平台商业源码许可（iHui Open Platform Commercial Source License）**  
即 Semi-Open Source License，以下简称「本许可」。

### 4.2 核心条款摘要

- **您可以使用、修改与内部部署**获得的代码，用于：
  - 查看、学习与内部技术评估
  - 在自有业务系统内集成、修改与部署
  - 在约定范围内为您的最终用户提供服务

- **您不得**：
  - 将源码或基于源码的衍生作品向任何第三方再分发、出售或免费公开
  - 将本产品用于与智汇 AI 构成直接竞争的产品或服务
  - 移除或篡改版权声明与许可说明

- **完整法律文本**以购买时随附的 `LICENSE` 文件为准；若与本文档冲突，以 `LICENSE` 为准。

### 4.3 与常见开源协议的区别

- 本许可**不是** Apache 2.0、MIT、GPL 等标准开源协议。
- 本许可是**商业源码许可**：允许内部使用与修改，但**禁止再分发与转售**，以保护商业与知识产权。

## 六、快速开始（购买后）

1. 获取源码：从私有仓库克隆或下载 Release 包。
2. 阅读仓库内 `README.md`：环境要求（如 Node.js 版本）、安装依赖、构建与运行命令。

4. 构建与部署：例如 `npm run build`，将 `dist/` 部署到您的 Web 服务器或容器。

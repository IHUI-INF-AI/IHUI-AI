# 品牌名翻译规范

## 概述

本规范用于统一「智汇AI社区 (iHui AI)」项目在各语言中品牌名的处理方式，确保多语言版本在品牌呈现上的一致性与专业性。

- **项目名**：智汇AI社区 (iHui AI)
- **公司名**：吉林省爱智汇人工智能科技有限公司
- **支持语言目录**：`en`、`en-US`、`ja`、`ko`、`zh-CN`、`zh-TW`
- **适用范围**：`src/locales/` 下的所有 i18n 文件（包括 `full/` 与 `modules/` 子目录中按语言划分的 JSON 文件），以及前端页面、组件中所有面向用户展示的文本。

> - 「智汇 / 智匯」在非中文语言中统一使用英文缩写 `IHUI`，不再出现中文汉字。

## 翻译规则

1. **中文语言（zh-CN / zh-TW）**：保留中文原名，简体用「智汇」，繁体用「智匯」，其余品牌名按简繁习惯转换（如「百宝箱」/「百寶箱」、「元宝箱」/「元寶箱」、「扣子」/「扣子」、「智谱清言」/「智譜清言」）。
2. **非中文语言**：使用音译或英文缩写，具体对照见上方「品牌名对照表」。不得在非中文 i18n 文件中保留中文品牌名汉字。
3. **公司全称**：按各语言习惯翻译。
   - `zh-CN`：吉林省爱智汇人工智能科技有限公司
   - `zh-TW`：吉林省愛智匯人工智能科技有限公司
   - `ja`：吉林省愛智匯人工知能テクノロジー有限会社
   - `en` / `en-US`：Jilin Aizhihui Artificial Intelligence Technology Co., Ltd.
   - `ko`：지린성 아이지후이 인공지능 과기유한공사
4. **品牌名组合**：当品牌名与其他词组合（如「智汇AI社区」「智汇AI社」「智汇开放平台」）时，按各语言既定译法处理，组合中的品牌名部分遵循本规范对照表。
5. **新增品牌名**：新增品牌名时，必须先在本文档登记对照表，再同步到所有语言文件，避免出现遗漏或不一致。

## 注意事项

- 品牌名不应出现在非中文语言的 i18n 文件中（用对应翻译代替）。例如 `en.json`、`ja.json`、`ko.json` 中不应出现「智汇」「智匯」「智谱清言」「扣子」「百宝箱」「元宝箱」等中文汉字。
- 新增品牌名时，需在此文档登记并同步到所有语言文件（`en`、`en-US`、`ja`、`ko`、`zh-CN`、`zh-TW`）。
- 修改已有品牌名译法时，需同时更新本文档与全部语言文件，保持文档与代码一致。
- 翻译审校时建议使用 `scripts/check-i18n.ts` 等校验脚本扫描非中文语言文件中残留的中文品牌名，确保规范落地。
- 本规范仅约束品牌名，普通功能文案的翻译不在本规范范围内。

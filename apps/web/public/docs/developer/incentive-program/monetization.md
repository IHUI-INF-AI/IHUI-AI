# 智汇AI社区开发者变现指南

在智汇AI社区，开发者可通过 **上传智能体** 参与激励计划，实现"睡"后收入，从AIGC迈向AGI，以下是变现路径介绍：

## 一、参与入口与发布方式

- **入口**：通过小程序进入（海报有小程序入口标识）
- **发布**：
  - 个人发布：免费
  - 企业专属上传渠道：100元/月，可共享开源智能体用于交流学习

## 二、收益模式对比（与"扣子"平台）

| 维度       | 扣子                     | 智汇AI社区                |
|------------|--------------------------|-------------------------|
| 开发者收益 | 0收益                    | 用户订阅分成            |
| 开发者成本 | 承担流量费用、算力消耗    | 用户按需付费，开发者零成本 |
| 推广支持   | 无商业化扶持             | 流量扶持、变现指导      |

## 三、额外激励

- 凭点击量拿激励，轻松赚"奶茶钱"
- 介绍新开发者（上传新智能体并在团长群报名），可获5元激励

## 四、新手助力

若想参与但不会开发，可联系团长（扫描海报二维码），智汇AI社区作为扣子官方合作伙伴，助力搭建AI应用新范式，开发者与用户能近距离交流~

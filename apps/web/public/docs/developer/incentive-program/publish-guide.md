# 扣子智能体接入 AI丨智汇社 介绍

## 添加【AI丨智汇社区】为公共渠道

### 1.打开扣子（https://www.coze.cn/studio），点击快速开始

![](./images/wechat_2025-08-12_110547_067.png)

### 2.进入「工作空间」后，点击「发布管理」中右上角的「发布渠道管理」，再选择右上角「+发布渠道」中的「公共渠道

![](./images/FWFSbLo4IoHrGtxCh8Xc0HR5nhh.png)

### 3.将【AI丨智汇社区】添加至公共渠道

![](./images/wechat_2025-08-12_111104_590.png)

## 发布智能体

### 4.点击工作空间>项目开发>点击要发布的智能体

![](./images/wechat_2025-08-12_111956_628.png)

### 5.点击发布按钮

![](./images/wechat_2025-08-12_112017_518.png)

### 6.找到【智汇 AI社区】> 授权

![](./images/wechat_2025-08-12_112101_639.png)

### 7.接受验证码完成授权

![](./images/wechat_2025-08-12_112142_454.png)

## 查看上架情况

发布完成后可以在【发布管理】页面查看所有发布的智能体。

待通过**AI丨智汇社**审核后，智能体将在**AI丨智汇社**上架，可通过搜索栏搜索或点击底部「智能体」进入查看。

以上为扣子智能体创建发布操作指南，如有任何问题，与我们取得联系。

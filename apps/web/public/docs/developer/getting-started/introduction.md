# 开发者平台介绍

## 平台概述

智汇 AI开发者平台为开发者提供统一的API接口，支持多种AI能力集成，包括对话、图像生成、智能体调用等功能。

## 核心能力

### 🤖 AI对话能力
- 支持多种大语言模型
- 流式响应支持
- 上下文管理
- 多轮对话

### 🎨 图像生成
- 文本生成图像
- 图像编辑
- 图像风格转换

### 🎬 视频生成
- Kling AI视频生成
- One-Click视频生成
- 视频编辑和处理

### 👤 智能体系统
- 智能体调用
- 智能体管理
- 智能体市场

### 📁 文件管理
- 文件上传
- 文件存储
- 文件处理

## 快速开始

1. **注册开发者账号** - 访问开放平台注册
2. **创建应用** - 在控制台创建应用
3. **获取API密钥** - 生成API Token
4. **查看文档** - 阅读API文档
5. **开始集成** - 使用SDK或直接调用API

## 技术栈

- **API协议** - RESTful API
- **认证方式** - API Key（Header），支持 `Authorization: Bearer ihui_xxx` 或 `X-Api-Key: ihui_xxx` 两种方式
- **数据格式** - JSON
- **字段命名** - camelCase

## 支持的语言

- JavaScript/TypeScript
- Python
- cURL
- 其他支持HTTP的语言
